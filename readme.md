## 数学建模2022 C题解析
古代玻璃制品的成分分析与鉴别

### 目录结构

```text
2022C/
├─ C题/
│  ├─ C题.pdf
│  ├─ 表单1.csv
│  ├─ 表单2.csv
│  ├─ 表单3.csv
│  └─ 附件.xlsx
├─ preprocess.py
├─ Q1.py
├─ readme.md
├─ readxlsx.py
├─ requeirments
└─ 题解.md
```

- `C题/`: 原始题目与数据文件目录
- `C题/C题.pdf`: 题目 PDF 文档
- `C题/表单1.csv`、`表单2.csv`、`表单3.csv`: 预处理数据表
- `C题/附件.xlsx`: 原始数据
- `preprocess.py`: 数据预处理脚本
- `Q1.py`: 问题一分析与建模脚本
- `readxlsx.py`: Excel/CSV 读入工具脚本
- `requeirments`: 依赖列表文件（安装所需第三方库）
- `题解.md`: 题目解答与分析说明
- `readme.md`: 本项目说明
